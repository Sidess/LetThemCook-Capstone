AI not responding            chat.config.js     Check responses added
Styling looks wrong          index.css          Check Tailwind config
Button click does nothing    App.jsx handler    Check function exists
Changes not showing          Browser cache      Ctrl+Shift+Delete cache


INTEGRATION POINTS FOR BACKEND:

These are the ONLY files you need to touch:

1. src/app/services/recipeService.js
   └─ Replace searchRecipes() with your API call
   └─ Replace the mock implementation with fetch()

2. src/app/config/chat.config.js
   └─ Replace AI_FALLBACK_REPLIES with real responses
   └─ Or connect to your AI API here

3. (Optional) src/app/data/recipes.js
   └─ Fetch from your backend instead of importing local data

Everything else stays the same!
